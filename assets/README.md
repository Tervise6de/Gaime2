# Assets

Minimal, functional art only — this is a systems-heavy, low-art strategy game.

Scope:

- **icons/** — UI and resource icons, favicon. Prefer inline SVG so icons stay
  crisp, tiny, and tintable via CSS.
- No 3D models, no large sprite atlases, no audio for now.

Keep everything here small and version-control friendly (SVG over PNG where
possible). Anything that isn't UI chrome probably belongs in `/src/data` as a
definition, not here as an asset.
